Що таке PhantomPacts?

PhantomPacts — це унікальна система "пактів" (тимчасових контрактів) між гравцями. Призрачні свитки випадково з'являються у світі, дозволяючи гравцям укладати союзи різних типів. Але будьте обережні — порушення пакту призводить до серйозних наслідків!

📜

Призрачні Свитки

Свитки випадково з'являються у світі з красивими візуальними ефектами. Знаходьте їх та використовуйте для укладання пактів!

🤝

4 Типи Пактів

Мир, Альянс, Торгівля та Захист — обирайте відповідний тип союзу для різних ситуацій.

⚡

Система Покарань

Порушення пакту викликає фантомів, втрату предметів, негативні ефекти та навіть тимчасовий бан!

🔌

Інтеграції

Повна підтримка PlaceholderAPI та LuckPerms для розширеного функціоналу.

Основні можливості

🎮 Геймплей

  • 🔮 Динамічний спавн свитків — налаштовувана частота та локації появи
  • ⏰ Тимчасові контракти — пакти автоматично закінчуються через заданий час
  • 🛡️ Захист союзників — неможливість атакувати партнера по пакту
  • 👻 Привиди відплати — агресивні фантоми переслідують порушників

🎨 Візуальна складова

  • ✨ Красиві частинки при спавні та підборі свитків
  • 🌀 Анімація паріння свитків у повітрі
  • 💫 Голограми з назвами свитків
  • 🎵 Звукові ефекти для всіх дій

🔧 Технічні особливості

  • 📊 PlaceholderAPI — множина плейсхолдерів для інших плагінів
  • 🔐 LuckPerms інтеграція — автоматична видача прав при пактах
  • ⚙️ Повне налаштування — всі параметри в config.yml
  • 🌍 Мультисвітова підтримка — налаштування світів для спавну свитків
  • 💾 Збереження пактів — всі дані зберігаються при перезапуску

Встановлення

📋 Вимоги:
• Spigot/Paper 1.20+ (рекомендується Paper)
• Java 17 або вище
• PlaceholderAPI (опціонально)
• LuckPerms (опціонально)

Крок 1: Завантаження

Завантажте останню версію плагіна з GitHub

Крок 2: Встановлення

  1. Зупиніть сервер
  2. Помістіть PhantomPacts-1.0.0.jar у папку plugins/
  3. Запустіть сервер
  4. Плагін створить конфігурацію в plugins/PhantomPacts/

Крок 3: Налаштування (опціонально)

Відредагуйте config.yml під свої потреби та перезавантажте плагін:

/pactadmin reload
⚠️ Важливо: Після зміни конфігурації обов'язково використовуйте команду reload, інакше зміни не застосуються!

Команди

Команди для гравців

Команда Опис Право
/pact offer <гравець> <тип> Запропонувати пакт іншому гравцю phantompacts.use
/pact accept Прийняти пропозицію пакту phantompacts.use
/pact decline Відхилити пропозицію пакту phantompacts.use
/pact list Показати ваші активні пакти phantompacts.use
/pact info <гравець> Інформація про пакт з гравцем phantompacts.use
/pact types Список доступних типів пактів phantompacts.use
/pact break <гравець> Розірвати пакт (застосовує покарання) phantompacts.use

Адмін команди

Команда Опис Право
/pactadmin reload Перезавантажити конфігурацію phantompacts.admin
/pactadmin force <p1> <p2> <тип> Примусово створити пакт phantompacts.admin
/pactadmin clear <гравець|all> Очистити пакти гравця або всі phantompacts.admin
/pactadmin give <гравець> [к-сть] Видати призрачні свитки phantompacts.admin
/pactadmin spawn Заспавнити свиток на вашій позиції phantompacts.admin
/pactadmin info Інформація про плагін phantompacts.admin

Приклади використання

# Запропонувати пакт миру гравцю Steve на 24 години
/pact offer Steve peace

# Прийняти пропозицію
/pact accept

# Подивитися свої пакти
/pact list

# Видати 5 свитків гравцю
/pactadmin give Player123 5

Налаштування спавну свитків

scrolls:
  # Інтервал перевірки спавну (тіки, 20 = 1 секунда)
  spawn-check-interval: 6000
  
  # Шанс спавну при перевірці (0.0-1.0)
  spawn-chance: 0.3
  
  # Максимум свитків у світі одночасно
  max-scrolls: 10
  
  # Час життя свитка (секунди)
  lifetime: 300
  
  # Дозволені світи
  allowed-worlds:
    - world
    - world_nether
✨ Нова функція! Свитки тепер спавняться тільки в безпечних місцях:
• На поверхні біля природних блоків (дьорн, грязь, камінь)
• У печерах з достатнім простором
• НЕ у воді, лаві, повітрі або всередині блоків

PlaceholderAPI

Внутрішні плейсхолдери плагіна

Плейсхолдер Опис
%phantompacts_pacts_count% Кількість активних пактів гравця
%phantompacts_has_pact% Чи є активні пакти (true/false)
%phantompacts_partners% Список партнерів по пактах
%phantompacts_scrolls_active% Кількість активних свитків у світі
%phantompacts_pact_with_<гравець>% Чи є пакт з конкретним гравцем
%phantompacts_pact_type_<гравець>% Тип пакту з гравцем
%phantompacts_pact_time_<гравець>% Час що залишився до кінця пакту
%phantompacts_in_pact_group% Чи належить до LuckPerms групи пактів

Використання в повідомленнях

Плагін підтримує будь-які плейсхолдери з PlaceholderAPI у повідомленні про підбір свитка:

# Базовий варіант
scroll-pickup-broadcast: "&d&l✦ &e%player_name% &aзнайшов свиток!"

# З префіксом LuckPerms
scroll-pickup-broadcast: "%luckperms_prefix%%player_name% &aзнайшов свиток!"

# З детальною інформацією
scroll-pickup-broadcast: "&e%player_displayname% &7[%player_world%] &aзнайшов свиток на &e%coords%"
💡 Порада: Якщо PlaceholderAPI не встановлено, плагін буде працювати з базовими плейсхолдерами (%player%, %coords%, %world%)

Система покарань

При порушенні пакту застосовуються наступні покарання:

👻

Привиди Відплати

5 агресивних фантомів з'являються та атакують порушника

💎

Втрата Предметів

30% випадкових предметів випадають з інвентаря

🧪

Негативні Ефекти

Висушування, слабкість та уповільнення

⭐

Втрата Досвіду

Мінус 10 рівнів досвіду

Налаштування покарань

punishments:
  spawn-phantoms:
    enabled: true
    amount: 5
  
  drop-items:
    enabled: true
    percentage: 30
  
  effects:
    enabled: true
    effects-list:
      - "WITHER:200:1"
      - "WEAKNESS:400:1"
      - "SLOWNESS:600:2"
  
  temp-ban:
    enabled: false
    duration: 3600  # 1 година
  
  remove-exp:
    enabled: true
    levels: 10
